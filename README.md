# Свадебный сайт Романа и Анны




## Telegram RSVP
Анкета отправляет ответы через Netlify Function: `netlify/functions/send-rsvp.js`.

В настройках Netlify добавьте переменные окружения:
- `TELEGRAM_BOT_TOKEN` — токен бота от @BotFather
- `TELEGRAM_CHAT_ID` — ID чата/канала/личного диалога, куда слать ответы

## Публикация
Загрузите папку проекта на Netlify. Для локального запуска можно просто открыть `index.html`, но отправка в Telegram будет работать только через серверную функцию.
